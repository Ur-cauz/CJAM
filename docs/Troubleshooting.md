# Troubleshooting

## ESP32 Doesn't Boot

- Check 3V3
- Check GND
- Check USB Cable

---

## OLED Doesn't Turn On

- Verify SDA
- Verify SCL
- Check I²C Address

---

## NRF Doesn't Respond

- Verify SPI Wiring
- Check CE
- Check CSN
- Verify Capacitors

---

## PCB Doesn't Work

- Inspect for shorts
- Measure 3V3 Rail
- Reflow suspicious solder joints
