# Assembly Guide

This guide walks through assembling the CJAM board.

## Required Components

| Qty | Component |
|----:|-----------|
| 1 | ESP32 DevKit V1 |
| 3 | NRF24L01+ Modules |
| 3 | 100µF Capacitors |
| 3 | 0.1µF Capacitors |
| 2 | 4.7kΩ Resistors |
| 1 | 1kΩ Resistor |
| 1 | LED |
| 1 | Pushbutton |
| 1 | 0.96" OLED |

---

## Recommended Tools

- Soldering Iron
- Leaded Solder
- Flush Cutters
- Tweezers
- Multimeter
- Flux (Highly Recommended)

---

## Assembly Order

1. Resistors
2. Ceramic Capacitors
3. Electrolytic Capacitors
4. LED
5. Push Button
6. OLED Header
7. NRF Headers
8. ESP32 Headers

---

## Before Powering On

- Check capacitor polarity.
- Inspect solder joints.
- Verify there are no shorts between 3V3 and GND.
- Verify ESP32 orientation.
